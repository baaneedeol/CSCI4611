# CSCI 4611 – Computer Graphics

## Course Overview

Computer graphics is an exciting and rapidly evolving field within computer science, with a strong impact on art, design, education, games, movies, science, and medicine. This course focuses on the tools and techniques used to build interactive graphics applications and games. Emphasis is placed on **hands-on programming**, modern **graphics toolkits**, and **C/C++-based development**, along with an understanding of the mathematical and physical principles behind computer graphics.

Throughout the course, students explore how interactive graphics systems are designed, implemented, and optimized, while also considering the broader history, future, and social impact of computer game technologies.

---

## Core Topics

* Event loops and user interaction
* Rendering and animation pipelines
* Polygonal models and mesh representations
* Texturing and realism tradeoffs
* Physical simulation and animation techniques
* Lighting, shading, and GPU programming
* Cameras, rays, and 3D user interfaces

By completing the assignments and modules, students gain experience building interactive, visually rich applications while understanding the core principles that drive real-world graphics systems.
